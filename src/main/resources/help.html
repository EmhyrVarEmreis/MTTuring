Opis klas pakietu pl.morecraft.studia.mt.rw.core

Klasa Przejscie

Obiekt tej klasy jest pojedynczym przejściem. Zawiera on pola:

  • stateA - obecny stan MT;
  • stateB - stan MT po przejściu;
  • symbolA - obecny symbol na taśmie;
  • symbolB - symbol do zapisania na taśmie;
  • direction - prawo/lewo - kierunek przesunięcia głowicy;

Wartości tych pól to liczby typu Integer. Kod tej klasy razem z opisem JavaDoc znajduje się w źródłach programu w pl/morecraft/studia/mt/rw/core/Przejscie.java


Klasa MTRwT

Obiekt tej klasy jest rdzeniem MT. Wykorzystuje klasę Przejscie. Zawiera pola:

  • tasma - taśma na której pracuje MT;
  • stanObecny - stan MT;
  • pozycja - pozycja na taśmie wskazywana przez głowicę MT;
  • przejscia - lista pzrejść - obiektów klasy Przejscie;
  • m, n - liczby;

Kod całej klasy razem z opisem JavaDoc znajduje się w źródłach programu w pl/morecraft/studia/mt/rw/core/Przejscie.java

Wykonywanie przejścia - robPrzejscie: 

	
	/**
	 * Metoda ta "robi" przejscie MT;
	 */
	public void robPrzejscie() {
		/* Sprawdza, czy MT nie zakończyła już działania */
		if (!isEnd) {
			/* Inkrementuje licznik przejść */
			licznikPrzejsc++;
			/* Wybiera odpowiednie przejście */
			Przejscie p = getPzrejscieFromAB();
			/* Sparwdza, czy ono istnieje */
			if (p != null) {
				/* Dokonuje przejścia */
				stanObecny = p.getStanB();
				tasma.set(pozycja, p.getSymbolB());
				pozycja += p.getStrona();
			}
			/* Sprawdza, czy MT jest w stanie końcowym lub nie ma kolejnych przejśc */
			isEnd = sprawdzCzyStanKoncowy();
		}
	}

Praca całościowa - makeAll: 

	
	/**
	 * Metoda ta dokonuje pracy MT;
	 */
	public void makeAll() {
		/* Przekazuje dane do GUI */
		przekazDane();
		
		/* Wykonuje pracę dopóki maszyna nie jest w stanie końcowym lub nie ma kolejnych przejśc */
		while (!isEnd) {
			
			/* Przejście */
			robPrzejscie();
			
			/* Usypia wątek na dany czas */
			try {
				Thread.sleep(czas);
			} catch (InterruptedException e) {
				e.printStackTrace();
				System.exit(2);
			}
			
			/* Przekazuje dane do GUI */
			przekazDane();
		}

		/* Przekazuje dane o wyniku końcowym do GUI */
		Main.app.setWerdykt(sprawdzCzyWynikPoprawny());
	}
